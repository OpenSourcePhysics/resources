Tracker Video Experiments

Tracker is a video analysis and modeling tool built on the Open Source Physics Java framework. These experiments showcase some of Tracker's unique features, including student-defined dynamic model simulations that synchronize with and are drawn on videos, "live" motion diagrams, interactive motion and force vectors, and a line profile tool for introductory spectroscopy.

  • Open tree folders to browse their contents
  • Click a tree node to learn more about it.
  • Double-click a green arrow to open a video experiment in Tracker.
