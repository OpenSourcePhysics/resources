Step 1: Explore this Collection

If you have not already done so, save a local copy of this Collection by choosing File|Save Collection As....

To explore the local (and therefore editable) Collection, open the Editor by clicking the Open Editor button. The editor displays a button bar for creating and organizing resource nodes and data fields for defining and describing the resources themselves.

Spend some time exploring the nodes, data fields and buttons to see how they behave. Note: editor button actions can also be executed by right-clicking a node and selecting the action from the popup menu. The Copy Node action is available by right-clicking any node in any collection, even when not editing.

Buttons:

  • Add Resource and Add Collection (available only when a collection node is selected): click to add a new resource or subcollection node to the selected node.
  • Copy Node and Cut Node: click to copy or cut the selected node to the clipboard. Note: the only way to delete a node is to cut it.
  • Paste Node (available only when a collection node is selected): click to add the clipboard node to the selected node.
  • Move Up and Move Down: click to change the position of the selected node.

Data fields:

  • Name: the node name displayed in the tree: click the field and enter the name directly.
  • Type: the type of resource: click the field and select from the dropdown list.
  • HTML File (optional): the URL path to the file displayed in the HTML pane when the node is selected: click the field and enter the path directly, or click the browse button to the right of the field and select the desired file with a file chooser. The path may be absolute or relative to the base URL. If no HTML file is specified, the DL Browser can display only very basic information about the resource.
  • Base URL (optional): the base URL used to resolve relative paths for the HTML and resource files: click the field and enter the path directly, or click the browse button to the right of the field and select the desired directory with a file chooser.
  • Resource (disabled for collection nodes): the URL path to the resource file: click the field and enter the path directly, or click the browse button to the right of the field and select the desired file with a file chooser. The path may be absolute or relative to the base URL.

Note: be sure to distinguish between HTML pages that are displayed in the DL Browser (defined in the editor's HTML File field) and those that are opened in the default web browser (defined in the editor's Resource field). The DL Browser's HTML pane is not a full-featured web browser, but it does support hyperlinks. Clicking a hyperlink such as this link to the Tracker Home site opens the target in the default web browser.
